## Problem Statement

You are given a dataset containing the following fields:

* `customerID`
* `orderID`
* `date`
* `transactionID`

The objective is to identify all customers who **have not performed any transactions during the previous month**.

A solution for this problem has been implemented in both SQL and PySpark.

You can find the implementations in the following files within this folder:

* `sql_solution.txt` – Contains the SQL-based solution
* `pyspark_solution.txt` – Contains the PySpark-based solution

Refer to the respective files for the complete query and implementation details.
